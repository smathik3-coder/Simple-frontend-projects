Neumorphic Web Calculator
A clean and minimalist Calculator web application built with a focus on Neumorphism UI design. This project is a fully functional calculator capable of performing basic arithmetic operations with a modern aesthetic.

=> Features
Basic Arithmetic: Supports addition, subtraction, multiplication, and division.

Clean UI: Designed using Neumorphic shadows for a 3D-effect.

Functionality: Includes specialized buttons for "Clear" (Cl) and "Delete" (Del) to manage input.

Responsive: Grid-based layout for consistent button spacing.

=> Tech Stack
HTML5: Structure of the calculator.

CSS3: Neumorphic styling and Grid layout.

JavaScript: Logic for calculations using the eval() function.
